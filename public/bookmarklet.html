How to Use

Step 1: Drag the button above to your browser's bookmarks bar
Step 2: Go to WordReference.com and search for a French word (e.g., "maison")
Step 3: Click the "Extract to Wordy" bookmark
Step 4: A popup will show the extracted data. Click "Copy to Clipboard"
Step 5: In Wordy, click "Import from WordReference" and paste the JSON

What Gets Extracted

  • Word: The French word (automatically lowercased)
  • Translation: The primary English translation
  • Example: Example sentence (if available)
  • Gender: Noun gender (le, la, les) if it's a noun

Example Output

{
  "word": "maison",
  "translation": "house",
  "example": "J'habite dans une grande maison.",
  "gender": "la"
}

Troubleshooting

Bookmarklet doesn't work?

  • Make sure you're on a WordReference page with a translation
  • Some browsers block bookmarklets - try copying the code manually
  • Check your browser console for error messages

Missing data?

  • Not all fields may be available for every word
  • You can fill in missing information manually in Wordy
